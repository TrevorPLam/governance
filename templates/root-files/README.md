# Root Files Templates

This directory contains templates for files that go in the repository root.

## Templates

Will include:
- **P0TODO.md** - Priority 0 (Critical) tasks template
- **P1TODO.md** - Priority 1 (High) tasks template
- **P2TODO.md** - Priority 2 (Normal) tasks template
- **P3TODO.md** - Priority 3 (Low) tasks template
- **CHANGELOG.md** - Changelog template
- **SECURITY.md** - Security policy template
- Other root-level documentation

## Update Policy

**Layer 1 - Custom**

These are templates only. Each repository customizes them fully and maintains them independently.

## Usage

1. Copy desired templates to repository root
2. Remove `.template` extension
3. Customize completely for your project
4. Maintain independently - no updates from governance repo
