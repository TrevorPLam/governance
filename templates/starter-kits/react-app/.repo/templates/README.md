# Document Templates

This directory contains standardized templates for project documents.

## Available Templates

Templates will include:
- README.md templates (various project types)
- Pull Request templates
- Issue templates
- Architecture Decision Records (ADR)
- Design documents
- API documentation
- User guides
- Contributing guides
- Code of conduct

## Update Policy

**Layer 2 - Updateable**

Templates can be updated from the governance repository. Projects can customize templates or add their own.

## Usage

1. Copy desired templates to your project
2. Customize with project-specific information
3. Use as starting point for new documents
4. Pull updates to get improved templates
