# Documentation Framework

This directory contains documentation standards and structures.

## Structure

### `standards/` - Documentation Standards
Standards and guidelines for:
- Documentation quality
- Writing style
- Code documentation
- API documentation
- Markdown conventions
- Diagram standards

### `adr/` - Architecture Decision Records
Templates and index for Architecture Decision Records (ADRs):
- ADR template
- ADR index
- Decision log
- Examples

## Update Policy

**Layer 2 - Updateable**

Documentation standards and templates can be updated from the governance repository.

## Usage

1. Follow standards in `standards/` directory
2. Use ADR template for architecture decisions
3. Maintain ADR index for discoverability
4. Pull updates to get improved standards
