# Training Materials

**Purpose:** Provide training guidance for governance adoption.

**Last Updated:** 2026-01-22  
**Version:** 1.0.0

---

## Training Tracks

### Beginner (2 hours)
- Governance overview
- Manifest configuration
- Running CLI commands
- Basic HITL and waiver workflows

### Intermediate (4 hours)
- Boundary rules and ADRs
- CI/CD integration
- Metrics collection and reporting
- Governance verification profiles

### Advanced (4 hours)
- Policy-as-code validators
- Migration and upgrade workflows
- Audit trail and compliance reporting

### Admin Track (3 hours)
- Rollout planning
- Versioning strategy
- Governance updates and migrations
- Incident response

---

## Delivery Formats
- Live workshop (remote or in-person)
- Self-paced reading + exercises
- Recorded walkthroughs

---

## Suggested Exercises
- Inject governance into a sample repo
- Fill `repo.manifest.yaml` correctly
- Run validation and verify reports
- Create and resolve a HITL item
