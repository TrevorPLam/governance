# Maintenance Plan

**Purpose:** Define long-term maintenance and support for the governance framework.

**Last Updated:** 2026-01-22  
**Version:** 1.0.0

---

## Release Cadence
- Patch releases: monthly
- Minor releases: quarterly
- Major releases: as needed

---

## Support Model
- Issues triaged weekly
- Security issues prioritized within 24 hours
- Backward compatibility by default

---

## Deprecation Policy
- Announce deprecations one release ahead
- Provide migration guidance
- Maintain compatibility when feasible

---

## Maintenance Checklist
- [ ] Review open issues monthly
- [ ] Refresh templates each quarter
- [ ] Validate example repos
- [ ] Update documentation
