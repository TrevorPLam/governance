# Governance Framework - Example

This example repository uses the AI-Native governance framework.

## Quick Links

- Policies: `policy/`
- Manifest: `repo.manifest.yaml`
- HITL: `policy/HITL.md`
- Waivers: `policy/WAIVERS.md`

## Usage

1. Run `governance validate`
2. Run `governance verify --profile=ci`
3. Follow policies for all changes
